## Learning Objectives

High Performance Computing (HPC) enables researchers and professionals to solve complex, data-intensive problems by harnessing the power of parallel processing. This session introduces the fundamentals of HPC, including its architecture, applications, and how to effectively access and utilise HPC systems for accelerated research and innovation. This session will include a number of practical hands-on exercises.

## Preparation and prerequisites

### Prerequisites

Prerequisites for the Introduction to High Performance Computing course: Charged laptop with Web browser, Installed SSH client. The course uses some basic elements of Python, C++ and C.

### Preparation

If you require assistance or further information with any of these please reach
out to us before the session.

## Installation and setup


### 1. Clone or fork the repository
Navigate to the location you want to install this repository on your system and
clone via https by running:
```
https://github.com/Cambridge-ICCS/hpc-summer-school
```
This will create a directory `hpc-summer-school` with the contents of this repository.

Please note that if you have a GitHub account and want to preserve any work you
do we suggest you first
[fork the repository](https://github.com/Cambridge-ICCS/hpc-summer-school/fork)
and then clone your fork.
This will allow you to push your changes and progress from the workshop back up
to your fork for future reference.

## Contribution Guidelines and Support

If you spot an issue with the materials please let us know by
[opening an issue](https://github.com/Cambridge-ICCS/hpc-summer-school/issues/new)
here on GitHub clearly describing the problem.

If you are able to fix an issue that you spot, or an
[existing open issue](https://github.com/Cambridge-ICCS/hpc-summer-school/issues),

Contributions from the community are welcome.
To contribute back to the repository please first
[fork the repository](https://github.com/Cambridge-ICCS/hpc-summer-school/fork),

If you would like support using these materials, adapting them to your needs,
or delivering them please get in touch either via GitHub or via
[ICCS](https://github.com/Cambridge-ICCS).
